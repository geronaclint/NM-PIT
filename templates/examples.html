Worked Examples

Two complete hand-solved problems demonstrating Richardson Extrapolation for first and second derivatives. All numeric values are shown to 4 decimal places.

Example 1

First Derivative of \( f(x) = \sin(x) \)

Given

  • Function: \( f(x) = \sin(x) \)
  • Point: \( x = \dfrac{\pi}{4} \approx 0.7854 \)
  • Step size: \( h = 0.1 \)
  • Exact derivative: \( f'(x) = \cos(x) \), so \( f'(\pi/4) = \cos(\pi/4) = \dfrac{\sqrt{2}}{2} \approx 0.7071 \)

Formula

Central difference for the first derivative:

\[ D_0^{(i)} = \frac{f(x + h_i) - f(x - h_i)}{2h_i} \]

Richardson combination:

\[ D_j^{(i)} = \frac{4^j \cdot D_{j-1}^{(i)} - D_{j-1}^{(i-1)}}{4^j - 1} \]

Step 1 — Central Differences (Column 0)

Row 0: \( h_0 = 0.1 \)

\[ D_0^{(0)} = \frac{\sin(0.8854) - \sin(0.6854)}{2(0.1)} = \frac{0.7742 - 0.6330}{0.2} = \frac{0.1412}{0.2} = 0.7059 \]

Row 1: \( h_1 = 0.05 \)

\[ D_0^{(1)} = \frac{\sin(0.8354) - \sin(0.7354)}{2(0.05)} = \frac{0.7416 - 0.6709}{0.1} = \frac{0.0707}{0.1} = 0.7068 \]

Row 2: \( h_2 = 0.025 \)

\[ D_0^{(2)} = \frac{\sin(0.8104) - \sin(0.7604)}{2(0.025)} = \frac{0.7246 - 0.6892}{0.05} = \frac{0.0354}{0.05} = 0.7070 \]

Row 3: \( h_3 = 0.0125 \)

\[ D_0^{(3)} = \frac{\sin(0.7979) - \sin(0.7729)}{2(0.0125)} = \frac{0.7159 - 0.6982}{0.025} = \frac{0.0177}{0.025} = 0.7071 \]

Step 2 — First Richardson Extrapolation (Column 1)

Using \( D_1^{(i)} = \dfrac{4 \cdot D_0^{(i)} - D_0^{(i-1)}}{3} \):

\[ D_1^{(1)} = \frac{4(0.7068) - 0.7059}{3} = \frac{2.8272 - 0.7059}{3} = \frac{2.1213}{3} = 0.7071 \] \[ D_1^{(2)} = \frac{4(0.7070) - 0.7068}{3} = \frac{2.8280 - 0.7068}{3} = \frac{2.1212}{3} = 0.7071 \] \[ D_1^{(3)} = \frac{4(0.7071) - 0.7070}{3} = \frac{2.8284 - 0.7070}{3} = \frac{2.1214}{3} = 0.7071 \]

Step 3 — Second Extrapolation (Column 2)

Using \( D_2^{(i)} = \dfrac{16 \cdot D_1^{(i)} - D_1^{(i-1)}}{15} \):

\[ D_2^{(2)} = \frac{16(0.7071) - 0.7071}{15} = \frac{11.3136 - 0.7071}{15} = 0.7071 \] \[ D_2^{(3)} = \frac{16(0.7071) - 0.7071}{15} = \frac{11.3136 - 0.7071}{15} = 0.7071 \]

Step 4 — Third Extrapolation (Column 3)

\[ D_3^{(3)} = \frac{64(0.7071) - 0.7071}{63} = \frac{45.2544 - 0.7071}{63} = 0.7071 \]

Final Answer

Best approximation: \( f'(\pi/4) \approx 0.7071 \)

Exact value: \( \cos(\pi/4) \approx 0.7071 \)

Absolute error: \( \approx 0.0000 \) (at 4-decimal display precision)

The Richardson Extrapolation improved the accuracy from about 3 correct digits (Column 0) to about 7 correct digits (Column 3).

Example 2

Second Derivative of \( f(x) = e^x \)

Given

  • Function: \( f(x) = e^x \)
  • Point: \( x = 1 \)
  • Step size: \( h = 0.1 \)
  • Exact value: \( f''(x) = e^x \), so \( f''(1) = e \approx 2.7183 \)

Formula

Central difference for the second derivative:

\[ D_0^{(i)} = \frac{f(x+h_i) - 2f(x) + f(x-h_i)}{h_i^2} \]

Step 1 — Central Differences (Column 0)

Row 0: \( h_0 = 0.1 \)

\[ D_0^{(0)} = \frac{e^{1.1} - 2e^{1} + e^{0.9}}{0.0100} = \frac{3.0042 - 5.4366 + 2.4596}{0.0100} = \frac{0.0272}{0.0100} = 2.7205 \]

Row 1: \( h_1 = 0.05 \)

\[ D_0^{(1)} = \frac{e^{1.05} - 2e^{1} + e^{0.95}}{0.0025} = \frac{2.8577 - 5.4366 + 2.5857}{0.0025} = \frac{0.0068}{0.0025} = 2.7188 \]

Row 2: \( h_2 = 0.025 \)

\[ D_0^{(2)} = \frac{e^{1.025} - 2e^{1} + e^{0.975}}{0.0006} = \frac{2.7871 - 5.4366 + 2.6512}{0.0006} = \frac{0.0017}{0.0006} = 2.7184 \]

Row 3: \( h_3 = 0.0125 \)

\[ D_0^{(3)} = \frac{e^{1.0125} - 2e^{1} + e^{0.9875}}{0.0002} = \frac{2.7525 - 5.4366 + 2.6845}{0.0002} = \frac{0.0004}{0.0002} = 2.7183 \]

Step 2 — First Richardson Extrapolation (Column 1)

\[ D_1^{(1)} = \frac{4(2.7188) - 2.7205}{3} = \frac{10.8752 - 2.7205}{3} = 2.7183 \] \[ D_1^{(2)} = \frac{4(2.7184) - 2.7188}{3} = \frac{10.8736 - 2.7188}{3} = 2.7183 \] \[ D_1^{(3)} = \frac{4(2.7183) - 2.7184}{3} = \frac{10.8732 - 2.7184}{3} = 2.7183 \]

Step 3 — Second Extrapolation (Column 2)

\[ D_2^{(2)} = \frac{16(2.7183) - 2.7183}{15} = 2.7183 \] \[ D_2^{(3)} = \frac{16(2.7183) - 2.7183}{15} = 2.7183 \]

Step 4 — Third Extrapolation (Column 3)

\[ D_3^{(3)} = \frac{64(2.7183) - 2.7183}{63} = 2.7183 \]
